### Frontend setup

1. Open terminal and navigate to the frontend folder
```bash
cd src/frontend
```

2. Then install the dependencies.
```bash
npm install
```


This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server on web:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

With electron:

```bash
npm run start:electron
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. With electron, it should also open a desktop window.
